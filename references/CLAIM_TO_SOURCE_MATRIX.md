## Status Definitions

| Status | Meaning |
|--------|---------|
| SOURCE_DOCUMENTED | Reported in official or scholarly sources; not independently reproduced |
| SOURCE_DOCUMENTED / PRIMARY_PROOF_PENDING | Announcement/source documented, but the current source set does not contain the primary proof |
| NOT_PROVIDED_BY_ORIGINAL_PROOF | The original proof/source explicitly does not provide this stronger claim |
| LOCALLY_REPRODUCED_EXACT | Verified by local exact arithmetic computation |
| AI_GENERATED_HYPOTHESIS | Generated by AI model; must be validated independently |
| REJECTED_OR_FAILED | Attempted and failed validation; preserved for transparency |
| INCONCLUSIVE | Validation attempted but result indeterminate |

## Upgrade Requirements for Asymptotic Claims

No asymptotic claim may be upgraded from source documented to locally
reproduced unless the repository contains an independent, reviewed
implementation or formal reconstruction of the relevant proof machinery and
tests. Finite baselines are not sufficient.
